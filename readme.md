\# GitLab Manager for NVDA



GitLab Manager is a powerful add-on for the NVDA (NonVisual Desktop Access) screen reader that brings GitLab repository management to your fingertips.



\## Features

\* \*\*List Repositories:\*\* View all your owned projects in a simple, accessible list.

\* \*\*Smart Navigation:\*\* Quickly open any repository's web page with a single click.

\* \*\*Advanced Creation:\*\* Create new repositories directly from NVDA.

&nbsp;   \* Set visibility (Private, Internal, or Public).

&nbsp;   \* Add project descriptions.

&nbsp;   \* Initialize with a README file.

&nbsp;   \* Choose from popular licenses (MIT, GNU GPL v3, Apache 2.0).

\* \*\*Secure Token Management:\*\* Uses GitLab Personal Access Tokens stored safely in NVDA configuration.



\## How to Use

1\. Obtain a \*\*Personal Access Token\*\* from your GitLab account (Settings > Access Tokens) with `api` scope.

2\. Open NVDA Menu > Tools > GitLab Manager.

3\. Enter your token when prompted.

4\. Use the interface to manage your projects.



\## Installation

Download the latest `.nvda-addon` file and open it to install. Restart NVDA to activate.
